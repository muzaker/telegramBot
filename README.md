# AzkarBOT

![logo](https://github.com/muzaker/telegramBot/raw/main/logo.png)

> A simple Bot telgram to azkar.

## Installation

Add `muzakerBOT` to your host.

with npm :

```bash
npm i
```

or with yarn:

```bash
yarn add
```

## how run

```bash
npm start
```

or

```bash
node main.js
```

or used pm2

https://pm2.keymetrics.io/

```bash
pm2 start main.js
```
